Directory Manager 2.0 Documentation: Bulk Changes

As an administrator or moderator (not available to others!), you can perform multiple simultaneous changes on more than one entry. This feature is called 'Bulk Change'.

This is done by running a search query, selecting (via checkboxes) which entries you wish to modify, and then specifying what fields should change -- adding a value (by 'Set'), replacing all values ('Set' with 'Replace All Values' checked), removing a value (via 'Unset/Remove' with specifying a value), or removing all values (via 'Unset/Remove' with no specified value).

You can also change the entry type for the given entries, such as to upgrade users to Samba 3. If you wish to change the entry type, you must explicitly set the 'Entry Type' drop-down list at the bottom.

Finally, you can delete entries from one click, by using the 'Delete Entries' button. CAUTION! Entries are NOT recoverable!
