Directory Manager 2.0 Documentation: Customizable Entry Types

Certain entry modules (i.e. users, groups) permit customizing what kind of entry you have by allowing customization of object classes (i.e. the 'entry type').

For example, a 'user' can be simply an 'address book entry', a 'UNIX user', or a 'Samba 3 + UNIX user'. This allows you to customize on an entry-by-entry basis which kind of entry they are.
Entry Type Drop-Down List:
You can ultimately mix and match entry types within the same tree. You CAN, however, limit which entry types are enabled via a configuration option. An entry type can be defined and changed ONLY by the administrator, and can be done via an entry's edit page near the top.
