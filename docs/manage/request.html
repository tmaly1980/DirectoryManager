Directory Manager 2.0 Documentation: Entry Requests

Requestors (and Admins as well) will have a button at the bottom of an add entry page saying 'Submit Request to Admins'. After submitting the new entry, an email will be sent to a configured address notifying of such. The entry IS NOT YET ENTERED INTO THE SYSTEM. This is done by the administrator at a later time, as described below.

Submit Add Request:

The administrators would then log into the site and click on 'Requests' in the respective module/tree, to then see this request (and others) for committing into the database. By clicking on an individual request, you are given the page to then commit it via the 'Save' button.
