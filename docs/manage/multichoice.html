Directory Manager 2.0 Documentation: Multiple-Choice Drop-Down Options

Some data fields can be given a pre-defined list of options in a drop-down menu. In such a case, there is also an 'Other:' option which allows you to use the corresponding text box to enter in an arbitrary value. Please refer to the Configuration Guide as to configuring these drop-downs.
