Directory Manager 2.0 Documentation: Entry Templates

Form fillout for adding entries can be simplified/expedited through templates. Templates specify pre-defined information for an entry.

There is a 'Default' template which will always load automatically when going to the add entry page.
Add Entry with Template:

Templates can be defined only by administrators. They are defined by filling out an add entry form (with whatever information you want to be auto-populated later on while loading), putting the name of the template (whether an existing one or a new name) in the 'Name' field, specifying whether the template shows up for just the tree (i.e. if there is more than one) or globally for all trees, and then clicking on 'Save As Template' at the bottom.

Saving a Template:

Templates are loaded by selecting their name in the 'Template' drop-down list and clicking on 'Load'.
