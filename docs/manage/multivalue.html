Directory Manager 2.0 Documentation: Multiple-Valued Fields

Some data fields allow multiple values. A rectangular, multi-line box will be provided, along with 'Add', 'Replace' and 'Remove' links to alter the values in this list. You also will be given the opportunity to change the order of these values being shown by highlighting a value and clicking on 'Up' or 'Down'.
