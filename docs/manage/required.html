Directory Manager 2.0 Documentation: Required Fields

Certain data fields are automatically required due to functionality. However, you can customize the list of required fields by adding MORE fields to require. This helps enforce that a minimum amount of information is entered when adding/modifying an entry. Required fields will then show up in red.

Please refer to the Configuration Guide as to configuring these required fields.
