Directory Manager 2.0 Documentation: Modifying Entires

Accessing the 'edit' page of an entry can be done in several ways (assuming that you're supposed to have access to an entry).

  • To access your own user account entry, make sure you're logged in and click on your name in the upper-left portion of the screen.
  • You can click on the 'Edit' link from an entry's 'View' page.
  • With the 'Implicit Edit' configuration option set, clicking on an entry from the 'Browse' or 'Search' page will go to the 'Edit' page.
Within the 'Edit' page, you are given the ability to edit ONLY relevent information. In specific, only administrators are allowed to alter internal system-specific information, such as UNIX/Samba account information. Common users can modify their personal, contact and employment information as well as their password, but they cannot alter their systems information such as their User ID number.

Deleting an entry is available ONLY to administrators, and is done by clicking on 'Delete' at the bottom-right of the edit page.
