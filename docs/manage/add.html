Directory Manager 2.0 Documentation: Adding Entries

Clicking on 'Add' in the navigational bar allows you to add new entries to the directory. Such ability is only granted to administrators and requestors.

You can then click on 'Add Entry' at the bottom of the page to submit the entry.

As a 'Requestor', you will only have the 'Submit Request to Admins' option available. Administrators may also submit entry requests (in such cases that there is some sort of approval process by another person). See the Entry Requests page for details regarding entry requests via requestors.
