Directory Manager 2.0 Documentation: Embedding Directory Manager

Directory Manager can be embedded into your own web pages from any web server that can access the Directory Manager web server. This is done through a CGI-based Javascript search form.

With Directory Manager, is an example web page called 'embedded.html' that shows the basic functionality of our embedded solution. To properly embed Directory Manager within your web site, follow the following tasks (this assumes you already have a web page you wish to insert the form into):

  1. Copy the content between the 'STEP 1' and 'END STEP 1' into your 'head' tag. This will insert any styles for the results table, as well as properly query the directory. Modify the parameters near the top to connect to the correct LDAP directory. Also, modify the styles to suit your needs (you can view 'templates/embedded_search.txt' to see how they are used).
  2. Copy the content between the 'STEP 2' and 'END STEP 2' into your HTML document body, to properly insert the search form and results.
