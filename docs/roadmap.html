Directory Manager 2.0 Documentation: Roadmap

Roadmap (Todo): Some things we are working on for our next release include:
  • More supported schemas:
    • Postfix, QMail, Courier SMTP/IMAP
    • Sudo
    • Sendmail
    • RADIUS
    • PAM Host Access Lists
    • DNS/DHCP (online or offline use)
  • Integration with Netscape servers/schemas (Calendar, Web Server, etc..)
  • Offline synchronization to an external /etc/passwd or NIS server
  • Scripts to facilitate NT-based entry management (via Samba)
  • Generating documentation mentioning account information for new hire employees
  • Passwords meeting certain security requirements (optional)
  • Generalized Access Control Management for external software with API's in Perl, PHP, Java, and C++
