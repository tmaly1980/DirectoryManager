Directory Manager 2.0 Documentation: Alphabetical List Results

By enabling this option, browse or search results aren't shown in a 'per page' fashion (if 'Entries Per Page' was enabled), but in such a way that entries get divided alphabetically rather than numerically. This will create a list of letters so that you can jump to those entries starting with each letter. Enabling this will make the 'Entries Per Page' option irrelevent.
