Directory Manager 2.0 Documentation: Users Module

The Users Module can have the following items configured on a tree-by-tree or global basis:

  • Enabled Entry Types
  • Default Entry Type
  • Bulk Migration
  • Starting User ID#
  • Ending User ID#
  • Create Home Directories
  • Home Directory Creation Settings
