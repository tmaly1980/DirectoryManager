Directory Manager 2.0 Documentation: Starting Group ID#

This specifies which group ID# to start with when adding group accounts. The actual group ID# will be auto-generated based upon what group ID#'s are available within LDAP, starting with THIS number. This should be in a separate range from local /etc/group entries to prevent conflicts.
