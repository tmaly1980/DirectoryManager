Directory Manager 2.0 Documentation: Authentication Tree

Which LDAP Directory Tree stores access control profiles under the 'ou=ACLGroups' node (as a master server). In other words, this tree houses the authoritative entries for users that use Directory Manager for administrative purposes. If there are no values, you must first create and initialize a tree via the 'Tree Settings' menu to populate the list of options.
