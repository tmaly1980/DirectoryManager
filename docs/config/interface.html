Directory Manager 2.0 Documentation: Interface Overview

The Directory Manager Configuration interface enables you to configure settings on a tree-by-tree basis.

It also permits 'global' settings, which allow for globally setting option across ALL trees without manually setting them for each one. These global settings also allow you to change things once and have it affect ALL servers immediately. To set an option Globally, simply select the 'Global' Scope from the drop-down menu, set the options, and click on 'Update'. To set tree-specific settings, simply change the 'Scope' drop-down menu to that specific tree.

The graphical interface is laid out such that you can switch between the settings of a particular directory tree or the global settings via a drop-down menu at the top-right. You can also add trees via the 'Add Tree' button next to it.

Switching between trees:

For each specific directory tree, you are given the ability to override the respective global settings. You do this by simply setting a value other than what it is currently set to.

To erase a tree-specific value and to fall back onto the global default, simply select the 'Inherit' checkbox for the respective configuration item and the click on 'Update'. Upon reload, the box should be checked meaning that it is inheriting the value, and the value displayed should be what is in the global settings.

You are given a list of menus via 'tabs', which group similar configuration options together and separate to maximize the ease-of-use. Having a specific tree selected will give you extra menus, such as 'Tree Settings' and 'Samba 3'. However, when Directory Manager gets first set up, none of the menus will be enabled until a tree is set up.

Configuration Menu Options:

The 'Access Control' and 'Licensing' tabs are considered part of the 'Global' configuration scope and will switch to that scope when clicked on.

* Note: It is REQUIRED to go through each and every menu and sub-menu to ENSURE proper configuration of Directory Manager. The values on the screen of each menu do not necessarily reflect what's in the configuration files until you click on 'Update'.
