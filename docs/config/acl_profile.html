Directory Manager 2.0 Documentation: Access Control Profiles

Access Control Profiles designate the groups that have administrative rights over a tree and/or a specific module of a tree. They include:

  • Profile Name: An alphanumeric name used to identify this profile.
  • Assigned Trees: A list of trees (internal abbreviations) that use this profile. You can also go to the tree's 'Tree Settings' and add a profile from there.
  • Administrative Group: The group that contains the DN's, to perform administrative access over the tree. It can be assigned to the entire tree, or for an individual module.
  • Moderator Group: The group that has moderator rights (can edit general information not vital to the functioning of the entry) for the entire network of LDAP servers. It can be assigned per tree or per module.
  • Requestor Group: The group that has request rights (can request that an entry be added, providing in the vital information that is not the decision of the administrators (mentioned above) to determine. It can be assigned per tree or per module.
