Directory Manager 2.0 Documentation: Bulk Migration

CAUTION! Use with care! This feature takes all entries within this module and migrates the entry type to the one specified here (as an alternate to the 'Bulk Change' feature within Directory Manager itself). This will ONLY work for the current tree. There is NO way to migrate all modules for all trees at once. This potentially could downgrade all the entries in your system to an unusable level!
