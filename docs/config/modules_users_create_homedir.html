Directory Manager 2.0 Documentation: Create Home Directories

Whether or not to create home directories on user creation. This gets done via an RPC daemon ('homedird.pl', provided with DM) running on the server hosting the home directories (which may or may not be the LDAP/DirectoryManager server itself).

See Home Directory Creation Overview for more more details.
