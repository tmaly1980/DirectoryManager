Directory Manager 2.0 Documentation: Access Control Groups

Access Control Groups specify which Distinguished Names (NOT usernames!) are granted a certain level of access (depending on how the group is designated in a profile). From the 'Access Control' Menu, you can click on the 'Click Here' link to modify the access control groups. You MUST log in as an Administrator of the Authentication Tree. In OTHER words, you MUST log in as the Root DN of the tree when you create your first access group.
