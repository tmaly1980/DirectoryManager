Directory Manager 2.0 Documentation: Starting User ID#

This specifies which user ID# to start with when adding user accounts. The actual user ID# will be auto-generated based upon what user ID#'s are available within LDAP, starting with THIS number. This should be in a separate range from local /etc/passwd accounts to prevent conflicts.
