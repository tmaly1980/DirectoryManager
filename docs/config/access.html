Directory Manager 2.0 Documentation: Accessing the Administration Site

To enter the Administrative site for initial configuration, open the Directory Manager website in your web browser. Prior to any configuration, you will automatically be presented with a link to the 'Administration' menu.

After the initial configuration, you can go to the Directory Manager site and click on the 'Administration' link (either on the bottom of the main page, or the Login page), OR go to /admin subdirectory under the URL that Directory Manager was installed on (i.e. if http://dir.malysoft.com/software/DirectoryManager, then the Admin site would be http://dir.malysoft.com/software/DirectoryManager/admin).

You will be asked for a username and password to be granted access, which is the username/password in the .htpasswd file set up during installation. Typically, the username is 'admin', and the password is by default 'DirectoryManager1', but is RECOMMENDED to be changed to something else via the htpasswd tool.
