Directory Manager 2.0 Documentation: Field Requirements

The data entry forms can have a customizable list of required fields in order to submit changes or a request. Fields that are required for functioning (such as 'cn') are automatically required. Here you can enter in PER TREE which fields should be required, typically when it makes some sort of business sense (such as requiring the corporate address book to be complete).
