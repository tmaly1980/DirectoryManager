Directory Manager 2.0 Documentation: Prerequisite Perl Modules

The required supplementary Perl modules include:
HTML::Parser 3.26 Net::SSLeay 1.25 (Optional, if using HTTP::Daemon::SSL or StartTLS) IO::Socket::SSL 0.96 (Optional, if using HTTP::Daemon::SSL or StartTLS) HTTP::Daemon::SSL 1.02 (Optional, if encrypting home directory creation requests) Digest::MD5 2.20 Net::LDAP 0.32 (aka perl-ldap) Digest::SHA1 2.10 Crypt::SmbHash 0.12 LWP::UserAgent 2.001

A script is provided to install these modules. When run, you will be asked a series of questions, including whether or not the current server is the home directory server or the modules should be installed locally to /var/www/DirectoryManager or system-wide (as a non-root user, it will ALWAYS install locally).

cd /var/www/DirectoryManager/bin ./install_modules.pl
