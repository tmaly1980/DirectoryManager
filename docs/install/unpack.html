Directory Manager 2.0 Documentation: Unpacking Directory Manager

  1. After downloading the software from the site, you should put the TGZ file into the parent directory that will host the web site, and then unzip it. For example:
    cp DirectoryManager.tgz /var/www cd /var/www tar zxvf DirectoryManager.tgz cd DirectoryManager
  2. From here (WITHIN the new DirectoryManager directory), enable executability (and other proper file permissions) for all CGI's and scripts:
    chmod a+rx . bin/* cgi-bin/*
  3. If you have a license file (named 'DirectoryManager-license.conf'), copy it to the etc/ directory (ignore this step for the Home Directory Server if different than your web server):
    cp DirectoryManager-license.conf /var/www/DirectoryManager/etc/
